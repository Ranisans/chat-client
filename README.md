# Simple Chat

## Steps To Run The Project
* clone repositories -

  chat-server
  ```bash
    git clone https://github.com/Ranisans/chat-server.git
  ```

  chat-client
  ```bash
    git clone https://github.com/Ranisans/chat-client.git
  ```

* run in both

  ```bash
    npm install
  ```
   or
  ```bash
    yarn
  ```
* then start the chat server in its folder
  ```
    npm start
  ```
  or
  ```
    yarn start
  ```

* and then start the chat client in its folder
  ```bash
    npm start
  ```
  or
  ```bash
    yarn start
  ```
  

*  go to http://localhost:3000/ in your browser
